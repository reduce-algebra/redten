Shift Operations

The process of raising and lowering indices via metric contractions (called shifting in REDTEN) is indicated by placing the @ symbol before the index element that is to be moved. This generates a contraction with the appropriate metric and creates a new object. Shift operations are processed by the function shift(), described in more detail in ยง3.1.
